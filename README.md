# chess-robot
This is a Chess Robot with APK Used, later Updated for a LEGO Chess Bot using the Power of a nother Device. With out these APIs it would not run: Stockfish (WASM / JavaScript Engine); chess.js; Web Worker API. Its not finished!

How to Start the Engine:

1. Open the link: https://danielatgermany.github.io/chess-robot/
2. Say if the engine or the player starts
3. Change the engine skill Level 1-20
4. Change the Counts of Threads and Hash your hardware can run proparly
5. Change the Engines Max Movetime (if the time is not needed, the engine will stop before the time runs out. 1000ms are 10seconds)
6. Now Push "Engine Starten" and the Engine will run and you can tippe in your move and the ai will make a good move for him. If the AI starts, push "Engine Starten" and the AI will automaticly start thinking and give you his good Play
